# InterFaiz
Interest Calculation Simulation Page
### Live here: https://inter-faiz.herokuapp.com/

## Getting Started

Steps in the bellow will allow you to open this site on your localhost.

### Prerequisites

_Node.JS and npm must be installed. Download and install them from [here](https://nodejs.org)._

### Installing

Follow these steps to run this project in your local computer.

```
$ https://github.com/geekysrm/react-loan-interest-calculator.git
$ cd react-loan-interest-calculator
$ npm i
```

Now, to run the project on port `3000`, run:

```
$ npm start
```

Go to `http://localhost:3000` to view the app.

## Built With

- [React.JS](https://reactjs.org/) -Frontend library used in the project.
- [Bootstrap](https://getbootstrap.com/) - Used for basic styling.
